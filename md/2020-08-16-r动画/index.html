R 动画

动画的本质是有n张图片进行拼接,到达欺骗人的眼睛.

在R中有常见的有两个包.

  • gifski 包, 搭配 rmd 文件使用,
  • gganimation 包

gifski 包

安装 gifski 包：

xfun::pkg_load2('gifski')

再用代码段选项 animation.hook='gifski'，如:

```{r, animation.hook='gifski'}
for (i in 1:2) {
  pie(c(i %% 2, 6), col = c('red', 'yellow'),
    labels = NA)
}
```

则会直接生成gif动画
